# استبدال الصور / Replace Images

## المطلوب / Required Images:

ضع صورك الحقيقية في مجلد `assets/products/` بالأسماء التالية:

1. **ceramic-tiles.jpg** - بلاط السيراميك
2. **marble-stone.jpg** - الرخام والحجر الطبيعي  
3. **stone-cladding.jpg** - كسوة الحجر
4. **lightweight-blocks.jpg** - البلوك الخفيف
5. **flooring-walls.jpg** - حلول الأرضيات والجدران
6. **supply-installation.jpg** - التوريد والتركيب

## فيديو الخلفية الرئيسية / Hero Background Video:

ضع فيديو الخلفية الرئيسية في مجلد `assets/`:
- **hero-background.mp4** - فيديو معرض أو منتجات فاخرة (1920x1080 بكسل، MP4)

## صورة قسم من نحن / About Section:

ضع صورة قسم من نحن في مجلد `assets/`:
- **about-image.jpg** - صورة معرض أو منتجات (1200x1500 بكسل)

## المواصفات / Specifications:

- الحجم الموصى به: 800x1000 بكسل
- الاتجاه: عمودي (Portrait)
- الصيغة: JPG أو PNG
- الجودة: عالية

## الخطوات / Steps:

1. احفظ صورك بالأسماء المذكورة أعلاه
2. ضعها في مجلد `assets/products/`
3. أعد تشغيل التطبيق: `npm run dev`
